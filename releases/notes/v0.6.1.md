# New Horizons OS Arduino v0.6.1

## Changes

- Fixed the OTA `check_update` and `lastStatusJson()` payloads so `changelog_url` is emitted as valid JSON instead of breaking the response envelope.
- Replaced the `ControlServer` and `OtaManager` ad-hoc JSON string parsers with shared helpers that correctly handle escaped quotes, backslashes, booleans, nested objects, and numeric fields.
- Reduced hot-path heap churn by reusing fixed scan averaging buffers and avoiding per-frame stream-host `String` copies in the UDP scan loop.
- Updated host-side regression coverage so OTA/control JSON contract checks verify the shared helper path instead of only checking for substring presence.
- Published the next firmware release metadata and release script default as `v0.6.1`.
