# New Horizons OS Arduino v0.5.15

## Changes

- Fixed OTA migration for legacy matrix layout config so saved `analog_pins` and `select_pins` are preserved after upgrading to newer firmware schema versions.
- Fixed FindMe post-OTA attach flow to require a fresh offer on the current boot before the device treats a gateway as attached, which avoids stale pre-OTA gateway state and repeated discover-only loops.
- Verified the firmware-side maintenance calibration command set remains intact for the new calibration workbench flow.
