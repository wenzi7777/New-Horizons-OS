# New Horizons OS Arduino v0.5.14

## Changes

- Added a real manual calibration module with session-based draft capture, saved profile persistence, and on-device calibration status reporting.
- Added `calibration_capture_all`, `calibration_capture_cell`, level dump, level delete, profile clear, enable/disable, and session begin/abort/commit control commands for the New Horizons calibration workbench.
- Connected the matrix scanner to the saved calibration profile so enabled calibration levels are applied directly in the scan path and exposed to the WebUI maintenance workbench.
