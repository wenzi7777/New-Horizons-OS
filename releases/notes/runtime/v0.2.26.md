# New Horizons Runtime v0.2.26

- Add native `vdboard.scan.pop_packet_into(buffer)` so normal streaming can reuse a preallocated MicroPython buffer.
- Move the scan task payload buffer from task stack to heap to reduce scan-task stack pressure on larger matrices.
- Release scan ring and payload storage if stream scratch allocation fails, improving recovery after ENOMEM.
- Add native scan memory stats for ring bytes, packet scratch bytes, filter/calibration bytes, scan payload bytes, and scan task stack high-water mark.

This runtime must be flashed manually; it cannot be installed by OS OTA alone.
