# New Horizons Runtime v0.3.0

- Emit UDP binary packet v2 with the full 6-byte device UID from the native scan core.
- Increase the native packet header to 20 bytes while keeping matrix payload values as float32.
- Keep fixed-capacity native scan buffers and ring buffer defaults for VD-CTL/R v1.0.F 2026.4.
- Match backend full MAC device identity to avoid duplicate short-ID devices.

This runtime must be flashed manually; it cannot be installed by OS OTA alone.
