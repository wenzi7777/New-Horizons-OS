# New Horizons Runtime v0.2.28

- Add native fixed-capacity scan frame, packet, ring, and filter buffers for the VD-CTL/R v1.0.F 2026.4 10x21 maximum matrix.
- Default native stream ring depth to 2 frames and reject values above the hard maximum of 8.
- Move scan execution into `vdboard.scan.service()` so the main loop drives scanning without a dedicated scan task stack.
- Keep packet header/template rebuild tied to layout and packet option changes, avoiding per-frame allocation.
- Preserve the existing binary packet layout and float32 sensor precision.
- Remove the runtime dependency on device-side MQTT modules.

This runtime must be flashed manually; it cannot be installed by OS OTA alone.
