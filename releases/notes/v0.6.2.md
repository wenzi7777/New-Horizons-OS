# New Horizons OS Arduino v0.6.2

## Changes

- Added a configurable packet ring buffer for matrix streaming, with `standard` mode using 3 frames and `extended` mode using 5 frames.
- Added persistent `stream_buffer` device configuration, a `set_stream_buffer` control command, and queue telemetry in `scan_health` and `status/query`.
- Updated the Desktop Diagnostics tab to control the stream buffer at runtime and display queue capacity, occupancy, and dropped-frame telemetry.
- Kept the previous direct-send path available when the stream buffer is disabled so ring-buffer behavior can be compared against the legacy mode without rebooting.
