# New Horizons OS Arduino v0.5.4

## Changes

- Added BQ25180 charge profile configuration through JSON control command `set_charge_profile`.
- Added WebUI Diagnostics controls for `compatible` and `fast` charging modes.
- Set `compatible` to 250mA charge current with 500mA input limit.
- Set `fast` to a conservative 300mA charge current with 500mA input limit.
- Kept VBAT regulation at 4.20V, termination at 10%, precharge at 20%, and fast-charge safety timer at 6 hours.
