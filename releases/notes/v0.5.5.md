# New Horizons OS Arduino v0.5.5

## Changes

- Added SPIFFS-backed device configuration at `/config/device.json` for matrix layout, scan timing, filter, IMU, external LED, and OLED settings.
- Added real SSD1306 128x32 driver with `off`, `auto`, and `enabled` modes. Default mode is `off`.
- Added dedicated external WS2812B controller for the three LEDs on GPIO12. Default mode is `off`; it is separate from the internal SK6812 status LED on GPIO11.
- Updated WebUI Settings and Terminal command schema for external LED and SSD1306 modes.
- Extended storage status with a `config` category so persistent device configuration is visible in flash diagnostics.
