# New Horizons OS Arduino v0.5.3

## Fixes

- Added `storage_status` for WebUI flash/storage diagnostics.
- Expanded `memory_status` with heap total, used, and min-free values for RAM usage bars.
- Returned explicit `filter.enabled` and `imu.enabled` state from configuration commands.

## Verification

- `python3 -m unittest discover -s tests`
- `arduino-cli compile --fqbn esp32:esp32:esp32s3:FlashSize=8M,PartitionScheme=default_8MB firmware/newhorizons_os`
