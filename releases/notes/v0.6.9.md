# New Horizons OS Arduino v0.6.9

## Changes

- Enabled boot-time auto OTA by default for fresh device configurations while preserving any existing saved OTA preference on upgraded devices.
- Locked in the scan-first main loop ordering so `scanAndStreamIfDue()` and `sendQueuedPacketIfAny()` run before Wi-Fi, FindMe, control, IMU, and display services, preventing the 3x3 matrix FPS regression from drifting down to about 48 fps at a 60 fps target.
- Added host-side regression coverage for the main loop scheduling contract and aligned the Arduino release script default version with `v0.6.9`.
