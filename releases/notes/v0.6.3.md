# New Horizons OS Arduino v0.6.3

## Changes

- Added UDP command reception on port 13250 (`ControlServer::serviceUdpCommand`): the device now reads incoming JSON control commands from the same UDP socket used for heartbeat and stream data.
- Commands received via UDP are immediately acknowledged with an ACK packet containing the sequence number, then a result packet is returned to the sender, enabling the Gateway's reliable-UDP retry loop to confirm delivery.
- Fixed `JsonUtils::findKeyColon` to skip malformed string tokens (non-string keys at the start of an object) rather than aborting, which prevented parsing nested `payload` objects inside the Gateway's command envelope.
