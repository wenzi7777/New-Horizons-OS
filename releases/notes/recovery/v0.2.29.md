# New Horizons Recovery v0.2.29

- Close TCP control cleanly before rebooting into OS, reducing stale sessions after OTA completion.
- Shorten TCP control connect timeout to avoid long repeated ETIMEDOUT periods after mode changes.
- Keep recovery update progress on TCP JSONL control while OTA file download continues through HTTP manifests.

Requires New Horizons Runtime v0.2.28 or newer.
