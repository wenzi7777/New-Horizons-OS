# New Horizons Recovery v0.4.1

- Add claim-aware New Horizons FindMe support in recovery mode.
- Add `findme_switch_gateway` so recovery can detach from the current Gateway and prefer a claimed Gateway during OTA or recovery maintenance.
- Report FindMe claim fields in recovery status for better Gateway diagnostics.
- Remove legacy `gateway_discovery` payloads from recovery updates.
- Keep Runtime at v0.3.0.

Requires New Horizons Runtime v0.3.0.
