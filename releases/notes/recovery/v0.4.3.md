# New Horizons Recovery v0.4.3

- Stabilize recovery-mode TCP control sends with the same outbound JSONL queue used by OS v0.4.3.
- Keep command results and OTA progress reliable while allowing transient socket backpressure to retry on the next poll.
- Add lightweight `memory_status` support for diagnostics during recovery.
- Reduce Wi-Fi portal storage imports unless portal storage details are explicitly requested.

Requires New Horizons Runtime v0.3.0.
