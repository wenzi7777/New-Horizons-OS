# v0.9.0

## Changes

### Pressure Calibration Runtime Cache

- Moved the expensive PCHIP curve preparation work out of the scan hot path.
- `Calibration::apply()` now uses cached per-sensor runtime curves instead of rebuilding vectors, sorting sample points, and recomputing slopes on every sensor read.
- Saved-profile mutators now rebuild the shared runtime cache when calibration data changes, so scan-time behavior stays equivalent while calibrated matrix FPS recovers substantially on larger layouts such as 10x14.

### Optional Stream Filter

- Added an optional firmware-side stream filter that matches the GCU placement and behavior: raw ADC sample -> median(1/3/5) -> IIR low-pass -> calibration mapping.
- The filter remains disabled by default and is intended for smoothing live stream frames only.
- Calibration capture helpers still read the raw path, so tare/level sampling is not polluted by the live filter state.
- The filter config surface now supports `enabled`, `median`, and `alpha`, while remaining backward-compatible with older payloads that only send `enabled`.
