# New Horizons OS Arduino v0.5.12

## Changes

- Restored BMI270 IMU runtime initialization for the new board without using BMM150 magnetometer paths.
- Replaced the previous hardcoded IMU `deferred` status with runtime `ready`, `error`, or `disabled` state reporting.
- Added IMU samples to Arduino UDP stream packets using the existing packet-v3 IMU flag and 7-float payload.
- Updated `set_imu` control responses to return the actual IMU runtime status and allow retrying initialization.
- Kept magnetometer CSV fields at zero on the new board while restoring nonzero `Gyro_*` and `Acc_*` data.
