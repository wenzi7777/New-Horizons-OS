# New Horizons OS Arduino v0.6.0

## Changes

- Added action-button soft power states with charger-aware `soft_off_battery` and `soft_off_charging` modes, plus control/status reporting through `power_set_state` and the `power` status payload.
- Suspended scan, Wi-Fi, OLED, external LED, and IMU services while soft-off is active so the board can stay charge-aware without leaving the main runtime fully on.
- Reworked the built-in `SK6812` indicator logic so stable states use low-brightness solid colors, transition states breathe, and command / OTA / warning signals overlay as short events. This also fixes the previously ineffective `OtaError` LED event.
- Updated the firmware release metadata and default Arduino release script version to `v0.6.0`.
