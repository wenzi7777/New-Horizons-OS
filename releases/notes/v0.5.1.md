# New Horizons OS Arduino v0.5.1

## Highlights

- Reorganized the repository into the active Arduino/hardware firmware layout:
  - `firmware/newhorizons_os/`
  - `firmware/scripts/`
  - `hardware/vd-ctl-r-v1.0f/`
- Added the VD-CTL/R v1.0.F hardware contract document.
- Added boot-window Wi-Fi provisioning: pressing the action button within 3 seconds of boot starts the setup AP.
- Kept automatic setup AP entry when no Wi-Fi credentials are stored.
- Published whole-firmware OTA artifacts and JSON manifests for Arduino firmware.

## OTA

- Firmware artifact: `releases/artifacts/newhorizons-os-v0.5.1.bin`
- Latest manifest: `releases/arduino-latest.json`
- Versioned manifest: `releases/arduino-v0.5.1.json`
