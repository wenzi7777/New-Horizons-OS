# v0.10.0

## Features

### Raw reading stream (rawADC)

- Added an optional stream of the pre-calibration raw reading (filtered
  millivolts — the same value emitted when the device is uncalibrated)
  alongside the calibrated level, so calibration can be diagnosed against the
  raw signal.
- Gated behind a new `set_raw_adc` control command and persisted as the
  `stream_raw_adc` device-config flag. **Off by default**; must be enabled
  manually from the desktop Device Settings page.
- Streamed via a new packet flag `0x08`: when enabled, the matrix payload
  carries a parallel float array of raw readings after the calibrated levels.
  When disabled the packet bytes are identical to prior firmware, so the change
  is fully backward compatible.
- `status`/`query` responses now report `stream_raw_adc` so the desktop can
  reflect the persisted state.
