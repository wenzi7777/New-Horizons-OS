# New Horizons OS Arduino v0.5.11

## Changes

- Refined the SSD1306 OLED pages so `live_status` now shows `NHOS vX.X.X`, device IP, gateway IP, FPS, and `RAM free/total`.
- Updated `sensor_snapshot` to show `Pts`, matrix `Grid`, last scan duration as `Scan`, and over-budget frame count as `Over budget`, removing the previous ambiguous `Last` and `Overrun` labels.
- Updated `recording_status` to show `Sent` and `Fail` counters with explicit `packets` labels plus current UDP latency in microseconds.
- Added scan-health row/column fields and regression coverage for the revised OLED content layout.

## Verification

- `python3 -m unittest discover -s tests`
- `arduino-cli compile --fqbn esp32:esp32:esp32s3:FlashSize=8M,PartitionScheme=default_8MB firmware/newhorizons_os --output-dir /private/tmp/newhorizons-ota-build`
- `VERSION=v0.5.11 firmware/scripts/build_arduino_release.sh`
- `firmware/scripts/generate_arduino_manifest.py --firmware releases/artifacts/newhorizons-os-v0.5.11.bin --output releases/arduino-v0.5.11.json --version v0.5.11 --base-url https://raw.githubusercontent.com/wenzi7777/New-Horizons-OS/v0.5.11/releases/artifacts`
