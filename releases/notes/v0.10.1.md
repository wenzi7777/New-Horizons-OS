# v0.10.1

## Fixes

### Safer default charging profile

- Changed the persisted-default charger profile from `balanced` to `slow`.
- Fresh boots and devices without a saved `charge_profile` now start at the
  `slow` profile, which maps to the 200 mA charging current on supported
  charger hardware.
- Existing devices that already saved another charge profile keep their stored
  preference.

## Verification

- Arduino scaffold and manifest regression tests.
- ESP32-S3 Arduino compile for the v1.0.F and GCU LTS release tracks.
