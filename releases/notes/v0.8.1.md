# v0.8.1

## Changes

### Pressure Calibration: PCHIP Interpolation Replaces Piecewise Linear

- Replaced piecewise linear interpolation (PLI) in `Calibration::apply()` with
  monotone cubic Hermite interpolation (PCHIP).
- The pressure–resistance relationship of the capacitive/resistive sensor material
  is strongly non-linear, with a steep climb in the low-force region. PLI
  introduces systematic errors proportional to local curvature × segment width²,
  which is largest exactly where force discrimination matters most.
- PCHIP reduces interpolation error by 1–2 orders of magnitude for the same set
  of calibration points, while preserving monotonicity (no overshoot or level
  reversal). For n=2 calibration points it reduces exactly to linear interpolation,
  so existing calibration data is fully backward-compatible with no re-calibration
  required.
- Circuit analysis of the ADC handler section (OPA4314AIPWR transimpedance
  amplifiers, RF = 5 kΩ): rawMv is already proportional to sensor conductance,
  so a conductance pre-transform would have provided no benefit. PCHIP was the
  correct and sufficient fix.
- Calibration session protocol, SPIFFS storage format, and all calibration
  commands are unchanged.
