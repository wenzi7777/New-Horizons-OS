# New Horizons OS v0.4.18

- Added resource governance for realtime streaming: UDP ENOMEM now pauses stream output, closes the socket, runs GC, and enters cooldown instead of retrying every loop.
- Kept TCP control/results ahead of realtime visualization traffic with priority result outbox handling.
- Reduced matrix layout and scan timing result payloads to scan-related state, stream state, and health data.
- Added `resource_state` and `stream_state` telemetry to status, memory status, scan health, and hello/status announce payloads.
- Added backend/WebUI stream diagnostics for dropped/coalesced visualization frames and WebSocket send FPS.
