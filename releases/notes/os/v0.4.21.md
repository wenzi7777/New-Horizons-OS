# New Horizons OS v0.4.21

- Split normal OS boot into a lightweight `app.py` wrapper and full `app_core.py` application module.
- Runs a low-memory Wi-Fi preconnect before loading the full application so STA auth/DHCP has recovery-like heap headroom.
- Keeps the full app behavior unchanged after preconnect; `App.setup()` sees the existing STA connection and continues normally.
- Retains the v0.4.20 optional IMU deferral and Wi-Fi timeout diagnostics.
