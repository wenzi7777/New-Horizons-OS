# New Horizons OS v0.4.19

- Added OS-level resource guard telemetry for `normal`, `watch`, `degraded`, and `critical` memory pressure states.
- Automatically decimates UDP visualization traffic under degraded heap pressure while keeping scan and TCP command/result paths active.
- Pauses UDP streaming under critical memory pressure, closes the UDP socket, clears buffers, runs GC, and enters stream cooldown.
- Added `send_every_n_frames` to `set_scan_timing` with a bounded `1..8` runtime setting; resource guard overrides are temporary and do not overwrite user configuration.
- Added Recovery write preflight protection so critical memory pressure in minimal mode returns `memory_pressure_preflight_failed` before downloads begin.
