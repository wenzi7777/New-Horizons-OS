# New Horizons OS v0.4.8

- Added an explicit `release_recovery_resources` command for Recovery updates.
- Recovery check/write now require the released-resource state instead of maintenance mode.
- Low-resource Recovery update mode keeps the device in normal mode while freeing scan, UDP, sensor, LED, and recorder resources.
- Status, memory status, scan health, and update results report `recovery_update_low_resource` so WebUI and terminal flows can gate check/write correctly.
