# New Horizons OS v0.4.20

- Deferred optional BMI270 IMU startup until scan is active and heap headroom is above the OS safety threshold.
- Kept BQ25180 battery startup guarded by a heap preflight so charging LED state can remain available without forcing critical heap pressure.
- Moved IMU/Battery driver imports out of the streaming service hot path to preserve memory for TCP control and status during boot.
- Hardened Wi-Fi STA success detection by accepting `STAT_GOT_IP` or a non-zero interface IP when `isconnected()` lags behind ESP-IDF events.
- Added timeout logs with STA status, IP, and free heap for future serial diagnostics.
