# New Horizons OS v0.3.1

- Report charger state in status and heartbeat payloads so WebUI can show Charging and Charge complete.
- Use the full 6-byte device UID for device name display and control identity.
- Keep WebUI recording failures from stopping live visualization updates.
- Keep Runtime at v0.3.0; this update only changes OS and Recovery payloads.

Requires New Horizons Runtime v0.3.0. Recovery v0.3.1 is recommended for full-MAC AP names.
