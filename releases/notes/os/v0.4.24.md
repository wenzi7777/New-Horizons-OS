# New Horizons OS v0.4.24

- Switch normal OS command/control from persistent TCP JSONL to UDP NHCP/1 TLV frames.
- Keep visualization stream on UDP and remove the normal OS TCP control module from the OS release package.
- Move FindMe discovery/offer traffic to NHCP TLV for the normal OS path.
- Reduce periodic status announcements to a lightweight scan/resource summary to avoid large diagnostic payload allocations.
- Preserve Recovery TCP JSONL OTA control for the current recovery update flow.
