# New Horizons OS v0.2.29

- Reduce periodic status announcement memory pressure by sending a compact heartbeat payload and keeping full system JSON for explicit status requests.
- Close TCP control cleanly before scheduled reboot so the backend can accept the next OS session faster.
- Shorten TCP control connect timeout to recover more quickly when the backend is restarting or holding a stale session.
- Keep the UDP data + TCP JSONL control transport introduced in v0.2.28.

Requires New Horizons Runtime v0.2.28 or newer.
