# New Horizons OS v0.4.15

- Fixes post-write Wi-Fi no-IP handling so OS no longer immediately hard-resets STA and bounces into an unreachable Recovery loop.
- Routes post-write network timeout to Recovery Wi-Fi setup mode, making the device visible through the setup AP when DHCP does not complete.
- Extends DHCP-pending observation before reconnect attempts to reduce auth/probe retry storms.
