# New Horizons OS v0.2.32

- Add runtime scan timing configuration via `set_scan_timing`.
- Allow custom positive `target_fps` and optional `settle_us` from terminal / backend commands.
- Add Settings > Pin Layout 60 FPS standard and 90 FPS extended presets in WebUI while keeping `settle_us` unchanged.
- Restart active native scan with health probe and rollback when timing changes cannot be applied safely.
- Include scan timing in status announce payloads so WebUI can display current FPS and settle time.

Requires New Horizons Runtime v0.2.28 or newer. Recovery remains compatible with v0.2.29.
