# New Horizons OS v0.4.14

- Prioritize Wi-Fi and WebUI recovery on the first OS boot after Recovery `write_os`.
- Defer local runtime services during post-write `wifi_link_no_ip` recovery, then start them after FindMe/TCP/UDP is online.
- Hard-reset STA on the first link-no-IP recovery attempt to avoid staying invisible after DHCP stalls.
- Reboot back to Recovery if post-write OS network recovery times out.
