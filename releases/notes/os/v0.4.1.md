# New Horizons OS v0.4.1

- Add FindMe gateway claim support through `findme_switch_gateway`, allowing the main backend to ask a device to re-run FindMe and prefer a specific Gateway.
- Keep claim state in device status, including preferred gateway, claim ID, expiry, and last claim error.
- Prefer matching FindMe claim offers during LAN discovery while still falling back to healthy accepted gateways when no claim is active.
- Remove legacy `matrix_scan_mock` and `gateway_discovery` payloads from OS updates.
- Keep Runtime at v0.3.0; this update changes OS and Recovery payloads.

Requires New Horizons Runtime v0.3.0. Recovery v0.4.1 is recommended so recovery-mode update and FindMe claim flows use the same protocol.
