# New Horizons OS v0.4.11

- Recovery update now enters a visible minimal system mode before checking or writing Recovery, instead of exposing it as a generic resource release.
- Recovery write publishes an immediate starting progress state and preserves intermediate file progress so WebUI does not jump directly from 0% to 100%.
- Status and diagnostics now report runtime services, and `service_control` can start, stop, or restart supported services from the task-manager diagnostics view.
- Web command fallback no longer treats stale completed update state as the result for a new write request.
