# New Horizons OS v0.4.17

- Reintroduced boot command safety guards for stale or wrong-mode reboot commands.
- Reintroduced visible minimal system mode for Recovery updates, with immediate Recovery write progress.
- Added lazy optional indicator loading: OLED defaults to off with off/auto/on modes, and external WS2812B defaults to off with off/on modes.
- Added offline standby state so offline recording remains available without starting scan/runtime hardware during boot.
- This release intentionally excludes the v0.4.12-v0.4.15 Wi-Fi recovery experiments.
