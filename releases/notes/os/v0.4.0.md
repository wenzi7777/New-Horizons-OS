# New Horizons OS v0.4.0

- Add New Horizons FindMe over UDP port 22346. Devices now only need Wi-Fi credentials; the local Gateway owns upstream server settings.
- Remove manual server host / TCP / UDP configuration from Wi-Fi setup and runtime settings.
- Add `findme_discover` command and `findme` status details for host, ports, gateway ID, last success, rejected gateways, and last error.
- Add FindMe no-gateway, rejected, and gateway-lost states to the system status LED and external indicator context.
- Remove matrix scan mock payloads; OS uses the real matrix scan path only.
- Keep Runtime at v0.3.0; this update changes OS and Recovery payloads.

Requires New Horizons Runtime v0.3.0. Recovery v0.4.0 is recommended so recovery-mode setup and OS update flows also use FindMe.
