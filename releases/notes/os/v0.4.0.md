# New Horizons OS v0.4.0

- Add LAN Gateway auto-discovery over UDP port 22346. Devices now only need Wi-Fi credentials; the local Gateway owns upstream server settings.
- Remove manual server host / TCP / UDP configuration from Wi-Fi setup and runtime settings.
- Add `discover_gateway` command and `gateway_discovery` status details for host, ports, gateway ID, last success, and last error.
- Add gateway discovery failure state to the system status LED and external indicator context.
- Keep Runtime at v0.3.0; this update changes OS and Recovery payloads.

Requires New Horizons Runtime v0.3.0. Recovery v0.4.0 is recommended so recovery-mode setup and OS update flows also use Gateway discovery.
