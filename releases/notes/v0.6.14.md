# v0.6.14

## Changes

### Allow File Read Without Maintenance Mode
- `file_read_begin` and `file_read_chunk` no longer require maintenance mode on the device.
- Reading files (log preview, file download) is a non-destructive operation and should be available in normal mode.
- Write and delete operations (`file_write_*`, `file_delete`) continue to require maintenance mode.
