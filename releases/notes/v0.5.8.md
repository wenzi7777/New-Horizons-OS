# New Horizons OS Arduino v0.5.8

## Changes

- Fixed external WS2812B ownership: GPIO12 is only driven by the external 3-LED controller, while GPIO11 remains the internal SK6812 status LED.
- Fixed external WS2812B enabled mode appearing to flash once and then turn off. Enabled stream-health and focus presets now keep a visible low-brightness state after the identify flash.
- Added external LED diagnostics to status: pin, count, initialized state, active preset, last show timestamp, and last error.
- Added an identify/test path for external LEDs so WebUI can trigger a 3-pixel sequence on GPIO12.
- Updated the Indicators settings flow so opening the tab requests live `status` and syncs persisted external WS2812B / SSD1306 settings instead of showing frontend defaults.
